Idle Upload Simulator

Welcome to Idle Upload Simulator!

In this game, your goal is to upload as many files as possible and earn credits.

Click the "Upload File" button to start uploading files. Each file has a random size and upload time.

Upgrade your network speed and credit value to earn more credits per upload.

Use the Save and Load buttons to save and load your game progress.
